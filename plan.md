# ScoreHub — 4종 시험 통합 채점 플랫폼

## 한마디 요약

> 응시자가 답안을 제출하면 AI가 채점하고, 플랜에 따라 분석 깊이가 달라지는 서비스.

---

## 시스템 위치

- **별개 사이트** (독립 프론트엔드)
- 백엔드: **Work Studio API** 사용 (DB 별도로 만들지 않음)
- 채점 로직: **Work Studio 플러그인** (`scoring-engine`)으로 모듈화

---

## 5종 시험 (B열 종목 연동)

| 종목 | 대상 | 핵심 평가 | 시간 |
|------|------|-----------|------|
| TESOL | 교사 | 수업설계, 평가도구, 학습자 상호작용 | 120분 |
| AI번역 | 번역사 | 정확성, 유창성, 맥락 적합성 | 60분 |
| AI프롬프트 | 기획자 | 정보성, 적절성, 효율성, 윤리, 보안 | 90분 |
| AI윤리 | 개발자 | 편향성, 투명성, 책임성, 프라이버시 | 60분 |
| ITT정통번역 | 통번역사 | 순차/동시통역, 음성번역 정확도 | 90분 |

> 5종목은 `_final_info.md` B열 종목과 동일. 급수 체계(교육/비즈니스/전문)도 연동.
> 채점 기준표는 종목마다 다르지만, **구조는 동일** (대분류 → 세부항목 → 배점)

> **채점 기준과 채점 엔진은 모든 플랜에서 동일.**
> 플랜 차이는 **분석 깊이**와 **전문가 개입 범위**뿐.

## 분류 체계 연동 (_final_info.md 참조)

| 분류축 | ScoreHub에서의 역할 |
|--------|---------------------|
| **A 소재** | 시험 문제의 도메인 태깅 (법률번역 문제, 요리 프롬프트 문제 등) |
| **B 종목** | 5종 시험 선택 = B열 종목 |
| **B 급수** | 시험 난이도 = B열 급수 (교육/비즈니스/전문 × 1~8급) |
| **C 사이트** | ScoreHub = C열 "시험" 카테고리 |
| **D 담당** | 전문가 첨삭 배정 시 D열 팀 연동 (문제은행팀, 강사팀) |
| **E 역할** | 향후 RBAC — 응시자(외부), 전문가 채점자(강사), 관리자(팀장) |

### 데이터 저장 (Work Studio DB)

| 테이블 | 용도 | 비고 |
|--------|------|------|
| `cbt_results` | 시험 결과 | 기존 테이블 활용 |
| `questions` | 문제은행 | 기존 테이블 활용 |
| `scorehub_submissions` | 답안 제출 (종목, 문제, 답안, 플랜) | 신규 |
| `scorehub_results` | AI 채점 결과 (점수, 세부항목, 코멘트) | 신규 |
| `scorehub_expert_reviews` | 전문가 첨삭 (3단계만) | 신규 |

## 사용자 플로우

```
[시험 선택] → [답안 입력 + 플랜 선택] → [AI 채점] → [결과 화면]
                                                         │
                                          ┌───────────────┼───────────────┐
                                          ▼               ▼               ▼
                                      1단계 무료      2단계 유료      3단계 프리미엄
                                      (즉시 표시)    (즉시 표시)    (AI 즉시 + 전문가 24h)
```

### 페이지 구성

| # | 페이지 | 설명 |
|---|--------|------|
| 1 | 시험 선택 | 5종 중 택 1 + 종목별 평가항목 안내 + 플랜 비교표 |
| 2 | 답안 제출 | 문제 표시 + 답안 입력(텍스트/파일) + 플랜 선택 + 결제 |
| 3 | 결과: 1단계 | 총점, 등급, 레이더, 대분류, 감점 Top3, 개선 Top3 |
| 4 | 결과: 2단계 | 1단계 전부 + 세부 채점 전체, 히트맵, 워터폴, 분포, PDF |
| 5 | 결과: 3단계 | 2단계 전부 + 전문가 첨삭, 수정 가이드, 매칭/상담 |

## 1단계 무료 — 결과 화면 구성

```
┌─────────────────────────────────────────────────┐
│  4-column 그리드                                  │
│                                                   │
│  COL1           COL2          COL3       COL4     │
│  ┌──────┐      ┌──────┐     ┌──────┐   ┌──────┐ │
│  │총점   │      │대분류 │     │감점   │   │개선   │ │
│  │스코어링│      │점수표 │     │Top 3  │   │Top 3  │ │
│  │       │      │      │     │      │   │      │ │
│  │레이더 │      │바차트 │     │기준표 │   │업그레│ │
│  │5각형  │      │달성률 │     │(접힘) │   │이드  │ │
│  │       │      │      │     │      │   │안내  │ │
│  │게이지 │      │      │     │      │   │      │ │
│  │5개    │      │      │     │      │   │      │ │
│  └──────┘      └──────┘     └──────┘   └──────┘ │
└─────────────────────────────────────────────────┘
```

### COL1: 총점 영역
- 스코어 링 (원형 게이지, 78/100)
- 등급 배지 (B+, 상위 32%)
- 레이더 차트 (5대 영역)
- 미니 게이지 5개 (영역별 달성률)
- 시험 정보 (시험명, 응시일, 문항유형, 소요시간)

### COL2: 대분류 분석
- 대분류 점수 테이블 (영역, 배점, 획득, 달성%, 상태)
- 영역별 달성률 바 차트

### COL3: 감점 분석 (요약)
- 감점 요인 랭킹 Top 3 (바 차트)
- 채점 기준표 원문 (아코디언, 접힌 상태)

### COL4: 행동 유도
- 개선 우선순위 Top 3 (항목 + 기대 상승 점수)
- 강점/약점 비율 스택 바
- **2단계 업그레이드 유도 카드**

## 기술 구현

### 프론트엔드
- **Vanilla JS + HTML** (work_studio 패턴, 빌드도구 없음)
- 또는 **Vite + vanilla TS** (가볍게)
- 4-column 반응형 그리드
- 차트: 순수 SVG (외부 라이브러리 없음, ScoreHub HTML 참조)

### 백엔드 (Work Studio API)

> 테이블은 위 "데이터 저장" 섹션 참조. 기존 `cbt_results`, `questions` 활용 + 신규 3개.

### AI 채점 엔진
- Work Studio의 `plugins/scoring-engine/` 모듈 사용
- 채점 기준표 (JSON) → AI API 호출 → 구조화된 결과 반환
- **모든 플랜에서 동일한 채점 실행** → 결과 중 플랜에 맞는 부분만 노출

### 플랜별 노출 제어

```js
// 채점 결과는 항상 전체가 있음. 플랜에 따라 보여주는 범위만 다름.
const result = await ScoringEngine.run(examType, answer);

if (plan === 'free') {
  show(result.summary);          // 총점, 등급, 대분류, Top3
  hide(result.details);          // 세부항목 잠금
  hide(result.charts.advanced);  // 고급 차트 잠금
}
if (plan === 'standard') {
  show(result.summary);
  show(result.details);          // 세부항목 전체
  show(result.charts.advanced);  // 고급 차트 전체
  enablePDF();
}
if (plan === 'premium') {
  show(result.summary);
  show(result.details);
  show(result.charts.advanced);
  enablePDF();
  requestExpertReview(result);   // 전문가 첨삭 요청
}
```

## 구현 우선순위

| 순서 | 작업 | 설명 |
|------|------|------|
| **Phase 1** | 시험선택 + 답안제출 + 1단계 무료 | MVP. AI 채점 동작 + 기본 결과 화면 |
| **Phase 2** | 2단계 유료 결과 화면 | 세부 차트, PDF, 분포 |
| **Phase 3** | 결제 연동 + 플랜 분기 | 1단계 → 2단계 업그레이드 플로우 |
| **Phase 4** | 3단계 전문가 첨삭 | 전문가 배정, 첨삭 UI, 상담 예약 |
